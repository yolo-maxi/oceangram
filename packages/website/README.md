# Oceangram Website

Landing page for Oceangram at oceangram.repo.box

## Features

- Hero section with clear value proposition
- Top 5 features showcase
- Installation instructions for VS Code and macOS
- Demo section (placeholder ready for GIF)
- "Built at repo.box" badge section
- Clean, responsive design

## Build

```bash
pnpm run build    # Copy src/ to dist/
pnpm run compile  # Alias for build
pnpm run dev      # Serve locally on :3001
```

## Deploy

Copy `dist/` contents to `/var/www/repo.box/subdomains/oceangram/`

## Demo Media

Replace `demo.svg` with an actual `demo.gif` showing:
- VS Code extension in action
- Tray app usage
- Telegram daemon features

Recommended dimensions: 800x400px
