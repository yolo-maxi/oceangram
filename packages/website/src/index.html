Get Started

VS Code Extension

  1. Download the latest .vsix from releases
  2. In VS Code: Ctrl+Shift+P → "Extensions: Install from VSIX"
  3. Select the downloaded file and install
  4. Open Oceangram panel from Activity Bar
Download Extension

macOS Tray App

  1. Clone the repository
  2. cd packages/tray && pnpm install
  3. pnpm run compile && pnpm run build:daemon
  4. pnpm start
